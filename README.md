# fwhigh.github.io and highonscience.com

Built with [Minimal Mistakes Jekyll theme](https://mmistakes.github.io/minimal-mistakes/).

## View locally

```bash
bundle exec jekyll serve --watch --drafts --incremental --livereload --future --open-url
```

View on http://127.0.0.1:4000/.

## Deploy

```bash
git commit -m "message" -a
git push
```
## Useful commands

```bash
bundle show minima
```
